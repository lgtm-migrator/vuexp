This is an overview of the most common usage of `<VxpLabel>`.

`VxpLabel` is a UI component that displays read-only text.

## Props

| Name     | Type    | Default | Web | Mobile |
| -------- | ------- | ------- | --- | ------ |
| text     | String  |         | ✔   | ✔      |
| textWrap | Boolean | `false` | ✔   | ✔      |
| visibile | Boolean | `true`  | ✔   | ✔      |
