### Label

> This is an overview of the most common usage of Label.

```html
<Label text="Label" />
```

`Label` is a UI component that displays read-only text.

#### Props

| Name     | Type    | Default | Web | Mobile |
| -------- | ------- | ------- | --- | ------ |
| text     | String  |         | ✔   | ✔      |
| textWrap | Boolean | `false` | ✔   | ✔      |
